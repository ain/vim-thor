# vim-thor

Vim plugin for [Thor](erikhuda/thor).

## Installation

Using [pathogen.vim](https://github.com/tpope/vim-pathogen) in your Vim
directory:

    git clone https://github.com/ain/vim-thor.git bundle/vim-thor

## Usage

For all Thor tasks:

    :Thor <task>
